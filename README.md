# 🚆 Integrated Track Management System (ITMS)
## 📌 Project Overview
The Integrated Track Management System (ITMS) is a smart railway safety system designed to detect track defects and obstacles in real-time using IR and Ultrasonic sensors.  
The system uploads live track status to the IoT cloud (ThingSpeak) for remote monitoring and automatic safety action.

This project aims to enhance railway safety by reducing accidents caused by track damage, cracks, and unexpected obstacles.

---

## 🎯 Objectives
- Detect railway track defects
- Detect obstacles on track
- Upload real-time data to IoT cloud
- Provide automatic train stopping mechanism
- Improve railway safety and monitoring

---

## ⚙️ Hardware Components Used
- Arduino Uno
- NodeMCU (ESP8266)
- Ultrasonic Sensor (HC-SR04)
- IR Sensor Module
- L298N Motor Driver
- Gear Motors
- LEDs (Red & Green Traffic Signal)
- Battery Power Supply
- Railway Track Prototype

---

## 🌐 IoT Platform
- ThingSpeak Cloud Dashboard
- Real-time Graph Visualization
- LED Widgets for Track Status Monitoring

---

## 🧠 Working Principle
- IR sensor detects track cracks or discontinuity.
- Ultrasonic sensor detects obstacles ahead of the train.
- Arduino processes sensor data and controls motor movement.
- NodeMCU uploads track status data to ThingSpeak cloud.
- If defect or obstacle detected → Train motor stops automatically and red signal turns ON.
